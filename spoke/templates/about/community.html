How to get involved with SayIt

SayIt is open source software. We built it for other people to use. If you'd like to get involved, here's where to start.

Social media

Say hello on Twitter or Facebook.

Mailing list

Implementing your own SayIt instance? Our mailing list community will help you with any problems that arise.

GitHub

Find the SayIt code in our GitHub repo. SayIt is licensed under the AGPL 3.0. We hope you'll pick it up, modify it if you want to, then fork it back to benefit the wider community.

The data standard we use is a subset of Akoma Ntoso – a standard developed specifically for use in parliaments and assemblies.

Volunteering

We're currently looking for volunteers to help us with SayIt. Whether you have coding skills, or transcripts you'd like to upload, mail us to find out more.
